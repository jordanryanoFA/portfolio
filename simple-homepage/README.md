# Simple Home Page Alarado - A challenge from devChallenge.io
modern, responsive landing page built using HTML, CSS, and JavaScript, featuring a clean UI, mobile navigation (hamburger menu), and a light/dark theme toggle.
This project focuses on layout structure, responsive design, and interactive UI elements.

# Features
1. Responsive Navigation
- Desktop navigation bar
- Mobile hamburger menu
- Smooth open/close animation
2. Light/Dark Mode Toggle
- Custom theme toggle switch
- Remembers user preference
3. Modern UI
- Clean typography
- Soft shadow cards
- Smooth hover animations
4. Fully Responsive
- Optimized for desktop, tablet, and mobile
- Hero layout automatically adapts

# Tech Stack
- HTML
- CSS
- JavaScript
- Google Fonts (Poppins)

# what I Learned
- Structuring a clean, scalable CSS architecture
- Making responsive navbar with a hamburger menu
- Using CSS variables to control themes
- Implementing dark mode using JavaScript
- Improving UI/UX with shadows, spacing, and typography
- Creating clean layouts using CSS Grid and Flexbox
- Mobile-first design principles
