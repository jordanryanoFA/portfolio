# Simple Detail Product Card

## Overview  
This is my implementation of a **Product Card** UI component from a devchallenges.io. 
The goal was to build a clean, responsive card using HTML and CSS, and deploy it live so I can show my end-to-end workflow.

- CSS styling for layout, spacing, image handling, typography and responsive design.  
- Responsive breakpoints to ensure the card looks good on mobile, tablet and desktop.
